# Home Dashboard

A small dashboard with weather, clock and timers. Loosely based on [Spate's Dashboard](https://github.com/spatie/dashboard.spatie.be/)

![Example](https://andreamswick.com/img/mockups/apps/home-dashboard.JPG)

## Usage

- Clone directory
- Serve!
